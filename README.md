# drillbit-test

This is a test repository for the project that will be developed as part of the RedHat Hackathon on 3/21/26.
The goal of the repository is to understand the tools that will be potentially used in the project:
1. Ramalama
2. Podman
3. Ollama
4. DNF/COPR API for Fedora
5. MCP
6. ChromaDB
